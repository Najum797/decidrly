Formula Reference

Monthly Mortgage Payment (PMT)

PMT = P × [r(1+r)ⁿ] / [(1+r)ⁿ − 1]
  • P = loan principal (home price − down payment)
  • r = monthly interest rate (annual rate ÷ 12 ÷ 100)
  • n = total number of monthly payments (years × 12)

Standard amortization formula. Each payment covers interest on outstanding balance plus a portion of principal.

Future Value (FV)

FV = PV × (1 + r)ⁿ
  • PV = present value (home price or down payment)
  • r = annual growth rate ÷ 100
  • n = number of years

Projects home value at horizon and future value of down payment if invested instead of used for purchase.

Present Value (PV)

PV = FV / (1 + r)ⁿ
  • FV = future cash value
  • r = discount rate per period (inflation ÷ 100)
  • n = number of years

Discounts future costs to today's money. Enables comparing buying vs. renting on equal footing.

Present Value of Annuity

PV = PMT × [1 − (1+r)⁻ⁿ] / r
  • PMT = periodic payment (monthly mortgage or rent)
  • r = monthly discount rate
  • n = total number of periods

Converts a stream of payments into a single present value — used for PV of all mortgage and rent payments.

Net Present Value (NPV)

NPV = Σ [ CFₜ / (1 + r)ᵗ ] for t = 1 to n
  • CFₜ = net cash flow in year t (rent saved minus buy costs)
  • r = discount rate per year
  • n = time horizon in years

Positive NPV = buying wins financially over the horizon in today's dollars.

Remaining Loan Balance

B = P × [(1+r)ⁿ − (1+r)ᵏ] / [(1+r)ⁿ − 1]
  • P = original principal
  • r = monthly interest rate
  • n = total payments over full term
  • k = payments made so far

Used to calculate equity at any point. Equity = Home Value (FV) − Remaining Loan Balance.

Monte Carlo Simulation

X = μ + σ × Z where Z ~ N(0,1)
  • μ = user-specified base rate (e.g. 5% appreciation)
  • σ = standard deviation: appreciation 2%, rent growth 1.5%, investment return 2.5%, maintenance 1%
  • Z = random normal draw via Box-Muller transform

1,000 scenarios are run. Results sorted; 10th, 50th, 90th percentiles reported as confidence bands.

Opportunity Cost of Down Payment

OC = DP × (1 + r_inv)ⁿ − DP
  • DP = down payment amount
  • r_inv = annual investment return ÷ 100
  • n = years

Capital in a down payment could be invested. This growth is subtracted from the renter's total cost for a fair comparison.
